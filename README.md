# 🗂 WorkDesk — Team Task Manager

A full-stack team task management app with role-based access (Admin/Member).

## Features
- Signup / Login (first user gets Admin role)
- Project & team management
- Task creation, assignment & status tracking
- Dashboard with stats, progress & overdue alerts
- Role-based access control

## Run Locally
```
npm install
npm run dev
```

## Deploy on Railway
1. Push to GitHub
2. New Project → Deploy from GitHub
3. Build Command: `npm run build`
4. Start Command: `npx serve dist`
5. Settings → Networking → Generate Domain
